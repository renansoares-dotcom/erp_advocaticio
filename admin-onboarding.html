🔑 Portal do Cliente

Crie credenciais de acesso para clientes acessarem o portal. A Service Role Key fica salva só neste navegador e nunca é enviada a ninguém além do Supabase.

⚠️ Onde encontrar: Supabase → Settings → API → "service_role" → Reveal. Nunca compartilhe essa chave.
